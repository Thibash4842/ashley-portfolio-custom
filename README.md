# Idlemind Portfolio

My personal portfolio website with a working contact form.

## Features
- Responsive design
- Contact form with email functionality
- Smooth page transitions with Swup

## Tech Stack
- HTML5, CSS3, JavaScript
- Node.js + Express backend
- Nodemailer for emails
- Swup for page transitions

## Live Demo
[https://github.com/Thibash4842]

## Contact Form Backend
The backend is hosted on Render and handles form submissions via Nodemailer.
